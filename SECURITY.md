# Security and Privacy

## Data privacy

This tool is designed to run locally in the browser. It does not require a server, login, or cloud database.

Project data entered by users is stored in local browser storage on the user’s own device. Users should export backups before clearing browser data or switching devices.

## Do not publish sensitive data

Do not upload:

- Real client names
- Real employee names
- Real budgets
- Real commercial data
- Credentials or API keys
- Internal project reports
- Private screenshots

## Reporting a security concern

Please open a GitHub issue with a high-level description and avoid posting sensitive exploit details publicly. If the repository owner provides a private contact method, use that instead.

## AI Prediction disclaimer

AI-style prediction is local, rule-based analytical logic. It does not call external AI services. Results are advisory and should be reviewed by a qualified project manager.
