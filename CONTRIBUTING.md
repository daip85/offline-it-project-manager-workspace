# Contributing

Thank you for considering a contribution.

This project is intended to help the IT project management community with a simple, offline, single-file workspace.

## Contribution principles

Please keep contributions:

- Generic and reusable
- Not specific to any company
- Free from proprietary templates or internal corporate formats
- Friendly for public community use
- Offline-capable where possible
- Easy to understand for project managers

## Good contribution areas

- Responsive UI improvements
- Accessibility improvements
- Additional chart types
- More project reports
- Better PMBOK / Agile formula explanations
- Better sample data
- Additional backup/export options
- Localization
- Bug fixes
- Performance improvements

## Before submitting

1. Test the HTML in a modern browser.
2. Confirm there are no secrets, tokens, credentials, or private URLs.
3. Confirm no real client, employee, vendor, or project data is included.
4. Keep the tool single-file unless a change is intentionally documented.
5. Update README or documentation if the feature changes user workflow.

## Suggested pull request format

```markdown
## What changed

## Why it is useful

## Screenshots / before-after

## Testing done

## Risks or limitations
```
