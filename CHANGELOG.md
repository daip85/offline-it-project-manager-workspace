# Changelog

## v1.0.0 — Initial Public Release

### Added

- Single-file offline HTML Project Manager workspace.
- Fixed Bid / Waterfall project mode.
- Agile Scrum / Time & Material project mode.
- Hybrid project mode.
- Project selector from top dropdown.
- All-project summary dashboard.
- Project-wise cockpit.
- Project-wise WBS and Gantt planning.
- Project-wise sprint board and backlog.
- Project-wise budget tracking.
- Project-wise team/resource management.
- Project-wise RAID and risk register.
- Project-wise reports and charting.
- Overall summary reports and charting.
- Project-wise AI Prediction.
- Overall AI Prediction.
- Theme options.
- Tooltips and guided help.
- Appendix with formulas.
- Codebase whitepaper section.
- Local browser storage.
- Import/export backup capability.
- Browser print/save-to-PDF workflow.

### Notes

This release is intended as a community-friendly productivity tool. It is not a replacement for enterprise PPM, contractual governance, or professional risk review.
